# traffic Light PCB

This is a PCB that's meant to mimic a traffic light! I made it because I thought it was cool and I wanted to learn more about timing and RC circuits!

Please keep in mind you need around 1% precise capacitors, resistors etc for exact accurate timing but normal parts do just fine: it may be off my 200-300 Milliseconds!

![3D PCB Image](https://cdn.hackclub.com/019d5874-5ee7-74c7-9828-98bfcc7216e3/20260404_13h22m11s_grim.png)

It uses RC timing and 2 Ne555p chip's that use Astable mode, this causes the led's to switch on and of just like a traffic light!
